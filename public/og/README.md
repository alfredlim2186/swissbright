# Open Graph Images

Place your Open Graph (social sharing) images here.

## Default Image

The default image is set in **Admin > SEO & Domain Settings**.

Current default: `/og/default.jpg`

## Recommended Specifications:
- **Format**: JPG or PNG
- **Size**: 1200 x 630 pixels (Facebook/Twitter standard)
- **File size**: Under 1MB for fast loading
- **Content**: Should include SweetB branding and be visually appealing

## How It's Used:
- When someone shares your SweetB link on social media (Facebook, Twitter, LinkedIn, etc.)
- The image specified here will appear in the preview card
- Helps increase click-through rates and brand recognition

## To Update:
1. Place your image in this folder (e.g., `default.jpg`)
2. Go to `/admin/settings` 
3. Update the "Default Open Graph Image Path" field to `/og/default.jpg`
4. Save settings

The change will take effect immediately across all pages.
