# Logo Files

Please place your logo files here:

## Required Files:

1. **sweetb-logo-peru.png** - The logo with "PERÚ" text (for main branding)
2. **sweetb-logo.png** - The logo without "PERÚ" text (for alternate uses)

## Recommended Specifications:
- **Format**: PNG with transparent background
- **Size**: At least 400px width for high-DPI displays
- **Color**: Already contains the gold "Sweet" and light pink "B" on transparent background

## Current Usage:
- Header (desktop & mobile)
- Footer
- Contact page
- About page

Once you add the logo files, the website will automatically display them instead of the text-based logo.
